Deskripsi Aplikasi

Aplikasi C Desa Sengonwetan adalah sistem informasi untuk mengelola data tanah dan persil di Desa Sengonwetan. Aplikasi ini memudahkan administrasi data kepemilikan tanah dan pencatatan data C Desa.

Fitur Utama
  • Pencatatan data C Desa
  • Pencarian dan filter data
  • Input data dengan scan dokumen
  • Export data ke PDF
  • Statistik data persil dan luas tanah
Kontak

Untuk informasi lebih lanjut, hubungi Kantor Desa Sengonwetan.
